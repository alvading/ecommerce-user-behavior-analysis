# 第 3–7 课：SQL 业务分析 | Lessons 3–7: SQL Business Analysis

计划按学习进度逐步加入以下经过实际验证的查询：

- 整体交易与销量 KPI / Overall transaction and sales-volume KPIs
- 年、季度和月度趋势 / Yearly, quarterly, and monthly trends
- 一级与细分类目表现 / Top-level and detailed category performance
- 用户购买频次与复购 / Customer frequency and repeat purchase
- 婴儿年龄、性别与购买行为 / Baby age, gender, and purchase behavior
- 用户分层 / Customer segmentation

查询不会提前批量生成。每完成一课，才把学习者理解并验证过的版本加入仓库。

Queries are not generated in bulk in advance. A query is added only after the
learner understands, runs, and validates it in the corresponding lesson.
