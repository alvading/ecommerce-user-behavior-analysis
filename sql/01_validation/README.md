# 第 2 课：数据质量验证 | Lesson 2: Data Quality Validation

本目录将保存行数、空值、重复、主键候选、日期范围、购买数量、性别值域及两表
关联覆盖率检查。查询必须由学习者执行，并将真实输出写入运行记录。

This directory will contain checks for row counts, nulls, duplicates, key
candidates, date ranges, purchase quantities, gender domains, and join coverage.
The learner must run each query and record the real output.
