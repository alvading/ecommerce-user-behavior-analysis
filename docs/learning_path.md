# SQL 引导式学习路线 | Guided SQL Learning Path

## 学习方式 | Learning method

每节课遵循同一闭环：定义业务问题 → 学习 SQL 知识点 → 预测输出 → 学习者编写并
执行 → 共同调试 → QA 验证 → 业务解读 → 双语记录 → 面试表达。

Every lesson follows: define the question, learn the SQL concept, predict the
output, learner writes and runs, debug together, validate, interpret, document
bilingually, and prepare an interview explanation.

## 课程顺序 | Lesson sequence

| 课次 | 主题 Topic | SQL 技能 SQL skills | 正式成果 Deliverable |
|---:|---|---|---|
| 1 | 数据与表结构 / Dataset and schema | 数据类型、键、事实与维度 | 建表设计 / Schema design |
| 2 | 导入与验证 / Import and validation | `CREATE`, import, `COUNT`, `DISTINCT`, `GROUP BY` | 运行记录 / Run record |
| 3 | 清洗与建模 / Cleaning and modeling | 日期转换、`CASE`, `JOIN`, QA | 干净事实/维度表 |
| 4 | KPI 与趋势 / KPIs and trends | 聚合、日期分组、比率、CTE | KPI 与趋势分析 |
| 5 | 商品与品类 / Product and category | 排名、窗口函数、分母控制 | 品类表现分析 |
| 6 | 复购与分群 / Repeat purchase and segments | 用户级聚合、分桶、`CASE` | 用户分析 |
| 7 | 婴儿属性 / Baby attributes | 多表 `JOIN`、年龄计算、覆盖率 | 年龄/性别分析 |
| 8 | Power BI 看板 / Dashboard | 数据集导出、指标对账 | 交互看板 |
| 9 | 报告与网站 / Report and website | 证据到洞察、故事表达 | 双语报告与案例页 |
